# PersonalPage
Página Pessoal desenvolvida com HTML, CSS e JavaScript — duas versões: Bootstrap e Bulma CSS. Projeto da disciplina Programação Web (UFLA).


# 🌐 Página Pessoal — Luana Peixoto Borges

Projeto desenvolvido para a disciplina **GAC116 - Programação Web** da Universidade Federal de Lavras (UFLA), como parte da Atividade Prática 2.

## 📋 Sobre o Projeto

Página pessoal desenvolvida com **HTML**, **CSS** e **JavaScript**, disponível em duas versões com frameworks CSS distintos, permitindo comparar diferentes abordagens de estilização.

- Layout responsivo (mobile, tablet e desktop)
- Tema claro e escuro com botão de alternância
- Preferência de tema salva no localStorage
- Animações ao rolar a página
- Seções: Hero, Sobre, Experiência, Interesses e Contato
